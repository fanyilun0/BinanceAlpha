**附录：量化评分模型细则**

1.  **交易量表现 (45%)**:
    *   **评分依据**: `(VOL/MC得分 + 24h交易量绝对值得分) / 2`
    *   **VOL/MC得分标准**: >0.5 (5分), 0.2-0.5 (4分), 0.1-0.2 (3分), 0.05-0.1 (2分), <0.05 (1分)。
    *   **24h交易量得分标准**: >$50M (5分), $20M-$50M (4分), $10M-$20M (3分), $5M-$10M (2分), <$5M (1分)。

2.  **价格稳定性 (40%)**:
    *   **评分依据**: 计算24h、7d、30d价格变化绝对值的平均值（`平均波动率 = (|24h%| + |7d%| + |30d%|) / 3`）。
    *   **得分标准**: 平均波动率 <5% (5分), 5%-10% (4分), 10%-20% (3分), 20%-30% (2分), >30% (1分)。

3.  **监管合规性 (5%)**:
    *   **评分依据**: 所有已上线币安Alpha的项目，默认通过初步筛查，给予基础分4分（满分5分）。若项目涉及明显高风险领域（如匿名混币），则会扣分。本次评估项目均无此情况。

4.  **代币分配与解锁 (10%)**:
    *   **评分依据**: `MC/FDV比例 * 5`。该值直接反映流通盘的占比和未来抛压，比例越高得分越高。MC/FDV=1时得满分5分。
