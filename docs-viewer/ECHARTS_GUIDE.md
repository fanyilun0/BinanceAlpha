# ECharts 图表使用指南

## 概述

本项目已成功集成 ECharts 图表库，提供了更强大的数据可视化功能和更好的用户交互体验。

## 主要特性

### 1. 默认选择 BSC/Base 链
- 页面加载时自动选择 BSC 和 Base 平台的数据
- 如果这些平台不存在，会自动选择所有可用平台
- 默认显示前 10 个符合条件的 Token

### 2. 全屏图表页面
- 访问路径: `/chart`
- 提供更大的展示区域
- 专注于数据分析和可视化
- 从主页点击"🚀 全屏图表页面"按钮可快速跳转

### 3. 强大的交互功能

#### 平台过滤
- 支持按区块链平台筛选 Token
- 点击平台标签可切换选择状态
- 支持全选/取消全选操作

#### 日期范围控制
- 快速选择: 最近 7/30/90 天或全部数据
- 滑块控制: 精确调整起始和结束日期
- 实时显示当前选择的日期范围

#### Token 选择
- 左侧列表显示所有可用 Token
- 点击 Token 名称切换选择状态
- 支持搜索功能，快速定位 Token
- 快速选择按钮: 前 5/10/20/30 个 Token
- 全选/清空按钮

#### 最小交易量过滤
- 滑块控制最小交易量阈值
- 实时过滤符合条件的 Token
- 显示当前阈值（单位：M 和 K）

### 4. ECharts 内置功能

#### 数据缩放
- **鼠标滚轮**: 在图表上滚动鼠标滚轮可缩放
- **拖拽选择**: 按住鼠标左键拖拽可选择区域放大
- **底部滑块**: 使用底部的数据缩放滑块调整显示范围

#### 工具箱
- **区域缩放**: 框选区域进行放大
- **还原**: 重置图表到初始状态
- **保存图片**: 将图表导出为 PNG 图片

#### 提示框
- 鼠标悬停在图表上显示详细数据
- 显示日期和所有 Token 的交易量
- 自动高亮当前选中的数据点

#### 图例
- 点击图例可显示/隐藏对应的 Token 曲线
- 支持滚动查看更多图例（当 Token 数量较多时）

## 使用步骤

### 方式一：在主页使用

1. 启动开发服务器: `npm run dev`
2. 访问 http://localhost:5173/
3. 点击"📈 图表"标签
4. 点击图表列表中的 `volume_time_series`
5. 在图表视图中，点击"🚀 全屏图表页面"按钮

### 方式二：直接访问图表页面

1. 启动开发服务器: `npm run dev`
2. 直接访问 http://localhost:5173/chart

## 操作技巧

### 快速筛选 BSC/Base 链数据
1. 页面加载后，BSC 和 Base 平台会自动选中
2. 如需调整，点击其他平台标签进行切换
3. 使用"全选平台"/"取消全选"按钮快速操作

### 分析特定时间段
1. 点击"最近 30 天"快速选择最近一个月的数据
2. 或使用滑块精确调整起始和结束日期
3. 观察图表变化，分析趋势

### 对比多个 Token
1. 在左侧列表中选择要对比的 Token
2. 使用"前 10"按钮快速选择前 10 个 Token
3. 在图表中观察它们的交易量趋势
4. 点击图例可临时隐藏某些 Token

### 导出图表
1. 点击工具箱中的"保存图片"按钮
2. 图表会自动下载为 PNG 格式
3. 文件名为 `token_volume_chart.png`

## 性能优化

- ECharts 使用 Canvas 渲染，性能优于 SVG
- 支持大数据量的渲染（已测试 200+ Token）

- 响应式设计，适配不同屏幕尺寸

## 常见问题

### Q: 为什么有些 Token 没有显示？
A: 可能是因为：
1. 最小交易量过滤器设置过高
2. 平台过滤器没有选择该 Token 所在的平台
3. 搜索框中有过滤条件

### Q: 如何重置所有设置？
A: 刷新页面即可恢复默认设置（BSC/Base 链，前 10 个 Token）

### Q: 图表加载很慢怎么办？
A: 
1. 减少选择的 Token 数量
2. 缩小日期范围
3. 提高最小交易量阈值

### Q: 如何查看某个 Token 的详细数据？
A: 
1. 点击图例隐藏其他 Token
2. 或使用搜索功能只选择该 Token
3. 鼠标悬停在图表上查看具体数值
