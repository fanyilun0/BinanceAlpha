# 更新日志

## 2025-12-03 - ECharts 集成和路由优化

### 主要更新

1. **使用 ECharts 替换 Chart.js**
   - 安装了 `echarts` 和 `vue-echarts` 依赖
   - 创建了新的 `VolumeChartEcharts.vue` 组件
   - ECharts 提供了更好的性能和更丰富的交互功能
   - 支持数据缩放、工具箱、图表导出等高级功能

2. **添加 Vue Router 支持**
   - 安装了 `vue-router` 依赖
   - 创建了路由配置文件 `src/router/index.js`
   - 实现了两个主要路由：
     - `/` - 主页（资源浏览）
     - `/chart` - 全屏图表页面

3. **创建独立的图表页面**
   - 新建 `ChartView.vue` 视图，提供全屏图表展示
   - 更好的展示区域，优化了用户体验
   - 可以从主页跳转到全屏图表页面

4. **优化交互体验**
   - **默认选择 BSC/Base 链**: 页面加载时自动选择 BSC 和 Base 平台的数据
   - **智能 Token 选择**: 默认选择前 10 个符合条件的 Token
   - **平台过滤**: 支持按区块链平台筛选 Token
   - **日期范围控制**: 支持快速选择最近 7/30/90 天或全部数据
   - **最小交易量过滤**: 可以设置最小交易量阈值来过滤 Token
   - **搜索功能**: 支持按 Token 名称搜索

5. **ECharts 特色功能**
   - **数据缩放**: 支持鼠标滚轮和拖拽缩放
   - **工具箱**: 提供数据缩放、重置、保存图片等工具
   - **优化的提示框**: 更美观的数据提示显示
   - **图例滚动**: 当 Token 数量较多时，图例支持滚动查看
   - **平滑曲线**: 使用平滑曲线展示趋势，视觉效果更好

### 技术架构改进

- **组件化**: 将主页逻辑拆分到 `Home.vue` 视图组件
- **路由管理**: 使用 Vue Router 管理页面导航
- **代码复用**: 保留了原有的 Chart.js 组件，便于对比和过渡

### 使用说明

1. **启动开发服务器**:
   ```bash
   npm run dev
   ```

2. **访问应用**:
   - 主页: http://localhost:5173/
   - 图表页面: http://localhost:5173/chart

3. **使用图表功能**:
   - 在主页点击"图表"标签查看图表
   - 点击"🚀 全屏图表页面"按钮进入专门的图表视图
   - 使用左侧的 Token 列表选择要展示的币种
   - 使用平台过滤器选择特定的区块链平台（默认选择 BSC 和 Base）
   - 调整日期范围和最小交易量来筛选数据
   - 使用 ECharts 工具箱进行缩放、重置、导出等操作
